# Renewal VO Service

## Prerequisites
- install [Node.JS](https://nodejs.org/en/download/)
- install [NPM](https://www.npmjs.com/)
- clone the [GitHub repository](https://github.com/voffice-id/renewal-vo-services)
- run `npm install` in the root directory
- copy .env.example to .env

## start
`npm run start`

## prettier check
`npm run prettier`

## prettier fix
`npm run prettier:fix`

## lint check
`npm run lint:check`

## lint fix
`npm run lint:fix`
